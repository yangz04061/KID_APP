# 幼儿英语启蒙 App 低保真原型说明（v1）

## 1. 文档目标

本文档用于描述首期产品的低保真原型结构，帮助在正式 UI 设计和开发前确认页面布局、核心元素和交互节奏。

## 2. 原型原则

- 每页只承载一个主要任务。
- 优先触控操作，不依赖文字阅读。
- 页面路径短，避免复杂跳转。
- 强化图片、语音、奖励反馈。
- 中文释义存在，但不抢儿童主视觉。

## 3. 页面原型

### 3.1 欢迎页

页面目标：

- 让孩子和家长快速进入应用。

线框说明：

```text
+--------------------------------------------------+
|                                                  |
|                [ 产品插画 / Logo ]               |
|                                                  |
|              幼儿英语启蒙小游戏                  |
|                                                  |
|                [ 开始探索按钮 ]                  |
|                                                  |
+--------------------------------------------------+
```

交互说明：

- 点击“开始探索按钮”进入首页。
- 整页不放其他干扰入口。

### 3.2 首页 / 主题选择页

页面目标：

- 让孩子快速选择喜欢的主题。

线框说明：

```text
+--------------------------------------------------+
| 标题：今天想玩什么？                 [家长入口] |
|                                                  |
|   [ 动物主题卡片 ]    [ 汽车主题卡片 ]          |
|                                                  |
|          [ 工程车主题卡片 ]                     |
|                                                  |
|         最近玩过：动物 / 星星数提示             |
+--------------------------------------------------+
```

交互说明：

- 点击主题卡片进入对应主题页。
- 点击家长入口进入家长区。

### 3.3 主题页

页面目标：

- 让孩子有进入一个主题世界的感觉。

线框说明：

```text
+--------------------------------------------------+
| [返回]              动物乐园                     |
|                                                  |
|          [ 大场景图：草地 / 动物园 ]            |
|                                                  |
|         今天会遇到 6 个动物朋友                 |
|                                                  |
|              [ 开始探索按钮 ]                   |
+--------------------------------------------------+
```

交互说明：

- 点击“开始探索按钮”进入单词预览页。

### 3.4 单词预览页

页面目标：

- 在正式游戏前快速建立图像和发音印象。

线框说明：

```text
+--------------------------------------------------+
| [返回]                 第 1 / 6                  |
|                                                  |
|                [ 大图：cat ]                     |
|                                                  |
|                     cat                          |
|                     猫                           |
|                                                  |
|        [重听发音]            [下一张/自动播放]   |
+--------------------------------------------------+
```

交互说明：

- 点击图片或“重听发音”播放英文发音。
- 点击“下一张”进入下一词卡。
- 最后一张结束后自动进入小游戏 1。

### 3.5 小游戏 1：听声音选图片

页面目标：

- 训练孩子从声音识别目标物体。

线框说明：

```text
+--------------------------------------------------+
| 第 1 关                              进度 1 / 5  |
|                                                  |
|            请找出：cat / 播放发音               |
|                                                  |
|   [图片 A]              [图片 B]                |
|                                                  |
|                [图片 C，可选]                   |
+--------------------------------------------------+
```

交互说明：

- 进入页面后自动播放发音。
- 孩子点击图片。
- 正确时出现星星和鼓励语音，随后进入下一题。
- 错误时轻提示并允许立即重试。

### 3.6 小游戏 2：图片配对

页面目标：

- 训练孩子建立图片与目标位置的关系。

线框说明：

```text
+--------------------------------------------------+
| 第 2 关                              进度 2 / 5  |
|                                                  |
|  把小动物送回家                                   |
|                                                  |
|  [cat图片] [dog图片] [lion图片]                  |
|                                                  |
|  [猫的家]   [狗的家]   [狮子的家]                |
+--------------------------------------------------+
```

交互说明：

- 支持拖拽，也支持点选图片后再点目标位。
- 配对成功后目标位点亮。

### 3.7 小游戏 3：找一找 / 点一点

页面目标：

- 通过场景探索增强趣味性。

线框说明：

```text
+--------------------------------------------------+
| 第 3 关                              进度 3 / 5  |
|                                                  |
|            Find the bus / 找到公交车             |
|                                                  |
|         [ 大场景图：马路 / 动物园 / 工地 ]      |
|                                                  |
|                [重听]      [跳过]               |
+--------------------------------------------------+
```

交互说明：

- 孩子点击场景中的目标对象。
- 找到后出现轻动画并进入下一题。

### 3.8 主题完成页

页面目标：

- 给孩子完成感，鼓励继续玩。

线框说明：

```text
+--------------------------------------------------+
|                 太棒了，你完成了                 |
|                                                  |
|               [ 徽章 / 星星动画 ]               |
|                                                  |
|         今天认识了 6 个动物单词                  |
|                                                  |
|      [ 再玩一次 ]          [ 返回首页 ]          |
+--------------------------------------------------+
```

交互说明：

- 点击“再玩一次”回到主题游戏。
- 点击“返回首页”回到主题选择页。

### 3.9 家长区

页面目标：

- 让家长看进度、调设置。

线框说明：

```text
+--------------------------------------------------+
| 家长区                                           |
|                                                  |
|  已学单词：12                                    |
|  最近常玩：动物                                  |
|  完成主题：1 / 3                                 |
|                                                  |
|  [背景音乐 开/关]                                |
|  [自动播放 开/关]                                |
|  [中文释义 显示/隐藏]                            |
|                                                  |
|                [ 返回首页 ]                      |
+--------------------------------------------------+
```

交互说明：

- 家长区优先采用列表式布局，减少视觉装饰。

## 4. 页面状态说明

### 4.1 正常状态

- 页面元素可点击。
- 发音正常播放。
- 进度正确显示。

### 4.2 反馈状态

- 点击正确：星星、短动画、鼓励语音。
- 点击错误：轻微抖动或柔和提示音，不显示明显失败文案。

### 4.3 空状态

- 首次使用时，首页不显示历史进度。
- 家长区首次显示“还没有学习记录，先去玩一玩吧”。

## 5. 原型评审重点

- 首页是否足够直观。
- 主题页是否有探索感。
- 单词预览是否过长。
- 3 类小游戏是否对 2 岁幼儿过于复杂。
- 中文释义是否影响儿童主界面。
- 家长入口是否足够隐蔽。

## 6. 下一步建议

1. 基于本线框说明绘制页面草图。
2. 确定每页按钮、卡片、弹层的统一组件规格。
3. 进入视觉设计前先做一次流程评审。
